# Prevent Close Tab

이 옵시디언 플러그인은 핀 상태의 탭과 사이드바에 있는 탭이 실수로 닫히는 것을 방지합니다.

## 주요 기능

- 핀 상태의 탭이 닫히는 것을 방지합니다
- 사이드바에 위치한 노트와 플러그인 탭이 닫히는 것을 방지합니다
- 다음 닫기 동작을 차단합니다:
  - 중간 마우스 클릭으로 탭 닫기
  - 탭 닫기 버튼(X) 클릭
  - 키보드 단축키(Ctrl+W, Cmd+W)를 사용한 탭 닫기

## 설치 방법

### 수동 설치

1. 릴리즈 페이지에서 최신 버전을 다운로드합니다
2. 다운로드한 zip 파일의 압축을 풉니다
3. `main.js`, `manifest.json`, `styles.css`(있는 경우) 파일을 옵시디언 볼트의 `.obsidian/plugins/prevent-close-tab` 폴더에 복사합니다
4. 옵시디언을 재시작하고 설정에서 플러그인을 활성화합니다

### BRAT를 통한 설치

1. [BRAT](https://github.com/TfTHacker/obsidian42-brat) 플러그인을 설치합니다
2. BRAT 설정에서 "Add Beta Plugin"을 클릭하고 이 저장소 URL을 입력합니다
3. BRAT가 플러그인을 설치한 후, 설정에서 활성화합니다

## 사용 방법

설치하고 활성화하기만 하면 됩니다. 추가 설정이 필요하지 않습니다.

- 핀 상태의 탭이나 사이드바에 있는 탭을 닫으려고 할 때 해당 작업이 자동으로 차단됩니다
- 일반 탭은 평소처럼 정상적으로 닫힙니다

## 작동 원리

이 플러그인은 다음과 같은 경우에 닫기 동작을 차단합니다:

| 탭 상태 | 닫기 동작 |
|---------|---------|
| 핀된 탭 | 차단됨 |
| 사이드바에 있는 탭 | 차단됨 |
| 일반 탭 | 정상 동작 |
